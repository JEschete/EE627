Use function glmfit to get the logistic regression coefficients
Use function glmval to get the logistic regression probability vector
Use perfcurve to get the ROC curve and AUC value.
 
You can use doc followed by the function name to get the help documents in Matlab
For example,  type "doc glmfit" , it will show the explanations of the function definition and usage example. 
 
Task 1:  Use predictor and response in the dataSet_1.mat to plot the ROC curve and obtain the AUC value.
 
Sample Code for Task 1:
 

clear all
close all

load dataSet_1.mat
factors=glmfit(predictor, response, 'binomial');
prob=glmval(factors, predictor, 'logit');
[X, Y, thre, AUC]=perfcurve(response, prob,1);
figure(1)
plot(X,Y)
[AUC]
